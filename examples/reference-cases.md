# Reference Cases

这个目录用于后续沉淀实际案例，建议每个案例至少保留：

- 参考图
- layout 提取结果
- element 提取结果
- 第一版实现
- diff 结果
- refine 后结果

建议的案例结构：

## Case Name

### Reference
- screenshot

### Layout Extraction
- module list
- implementation hierarchy

### Element Extraction
- element inventory

### First Pass Implementation
- page / component output

### Diff
- key mismatches

### Refine
- what changed
- what is still missing
